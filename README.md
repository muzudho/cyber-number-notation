# Addition of the last two rules

* Ends with o1

```plaintext
O0      # Bad
O0o1    # Ok

O1      # Ok
O2      # Bad
O2o1    # Ok
```

* o1 cannot be continuous

```plaintext
O1o1          # Bad
O1o1o1        # Of course it's bad
O1o2o1        # OK
O1o2o1o2o1    # OK
O1o1o1o2o1    # Of course it's bad
```
